# Hiver Frontend

This is the frontend starter scaffold for the Hiver application.
It is configured for cross-platform development with React Native / Expo and TypeScript,
including web support via `expo start --web`.

## Quick start

```bash
cd frontend
npm install
npm run start
```

## Available scripts

- `npm run start`: Launch Expo development tools
- `npm run web`: Run the app in the browser
- `npm run android`: Launch on Android emulator/device
- `npm run ios`: Launch on iOS simulator/device
- `npm run build`: Build web assets
- `npm run lint`: Run ESLint
- `npm run typecheck`: Run TypeScript checks

## Directory layout

- `App.tsx` — root application entry
- `components/` — reusable UI components
- `pages/` — screen/page components
- `constants/` — shared app constants
- `hooks/` — reusable hooks
- `lib/` — helpers and API clients
- `assets/` — static assets and placeholders
- `scripts/` — useful helper scripts
